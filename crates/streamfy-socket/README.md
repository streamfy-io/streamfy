# streamfy-protocol

Provide TCP socket wrapper for Streamfy protocol

# build streamfy send utility

build streamfy-send utility

```cargo build --all-features```

invoking streamfy-send utility to send api request sample data

```../target/debug/streamfy-send data/apirequest.bin```
